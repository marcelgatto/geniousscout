How it works

  1. Set up: Enter a name and pick a profile (Balanced / Quant / Ops / Policy).
  2. Play: The timer starts on the first task. Skip is allowed (counts as incorrect).
  3. Score: See subscores + composite. Heuristic percentiles for fun context.
  4. Share: Export JSON, print a PDF, or copy a link with results embedded.

What you get

  • Clear, single‑page results
  • Role‑aware weighting for different profiles
  • No signup, no data collection by default
  • Works offline once loaded

Ethics & limits

  • It’s a toy signal—not a diagnostic or selection tool.
  • Use alongside interviews and practical tasks.
  • Avoid high‑stakes decisions based on one score.
